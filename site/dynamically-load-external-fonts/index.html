Dynamically Load External Fonts

This paragraph is generated by the code below. It loads the Freehand font dynamically and applies it here. Press this button to see it in action:

HTML


<p id="text-to-update">
  This paragraph is generated by the code below.
  It loads the Freehand font dynamically and
  applies it here. Press this button to see it
  in action:
  <button id="load-trigger">Switch To Freehand</button>
</p>

JavaScript


const kickoff = () => {
    const loadButton = document.getElementById('load-trigger')
    loadButton.addEventListener('click', handleClick)
}

const handleClick = () => {

    const font = 'Freehand'
    
    const link = document.createElement('link')
    link.href = `https://fonts.googleapis.com/css2?family=${font}`
    link.rel = 'stylesheet'
    
    const head = document.getElementsByTagName('head')[0]
    head.appendChild(link)
    
    const text = document.getElementById('text-to-update')
    text.style.fontFamily = font
    
}

document.addEventListener('DOMContentLoaded', kickoff)

Details

The code works by creating a new <link> element and adding it into the <head> of the document. Then, it assigns the requested font face to the target css selector. The assignemnt happens before the font has been downloaded, but the browser applies it once the download is complete.
